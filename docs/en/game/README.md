# Game Spec Modules

This folder is the modular reading path for [`../game-spec-v1.md`](../game-spec-v1.md).

## Core foundations

- `01-overview.md`: product positioning, scope, core loop, and time rules
- `02-progression-and-combat.md`: progression, stats, combat rules, and skill-kit overview
- `03-world-systems.md`: equipment, economy, map, buildings, quests, dungeons, and arena

## Platform and delivery

- `04-bot-platform.md`: bot integration, public events, backend architecture, data model, and API quality
- `05-website-ops-and-delivery.md`: website requirements, observability, security, testing, launch, and delivery phases

## Deep-dive extensions

- `06-world-map-definition.md`: geography, route network, regional identity, and website map presentation
- `07-location-catalog-and-resource-definition.md`: location-by-location lore, facilities, dungeon links, and material outputs
- `08-seasonal-leveling-and-stat-framework.md`: seasonal progression cadence, XP curve, and base stat growth
- `09-equipment-dungeon-and-loot-framework.md`: equipment slots, dungeon gear families, and rating-based rewards
- `10-combat-system-framework.md`: turn flow, action resolution, status logic, AI, and battle log contracts
- `11-class-skill-system.md`: class skill pools, loadout rules, and auto-battle skill selection
- `12-dungeon-monster-and-difficulty-system.md`: monster tiers, room composition, boss placement, and difficulty scaling
- `13-dungeon-data-tables-and-template-spec.md`: table schema and validation rules for dungeon content authoring
- `14-first-batch-dungeon-balance-sheets.md`: first-batch dungeon compositions, monster stats, and validation checklist
- `15-battle-consumable-and-potion-system.md`: potion economy and in-dungeon sustain rules
- `16-battle-auto-resolution-flow.md`: automatic battle runtime flow from setup to settlement
- `17-combat-power-evaluation-and-preview-system.md`: combat power scoring and strength preview models
- `18-friend-and-bot-relationship-system.md`: friendship, assist permissions, and long-term bot relationships
- `19-bot-chat-and-channel-system.md`: world and regional chat channels for bot social signaling
