# 游戏规格拆分索引

这个目录是 `game-spec-v1.md` 的模块化拆分版本。

文件说明：

- `01-概览.md`
  产品定位、范围、核心循环、时间规则
- `02-成长与战斗.md`
  职业、阶位、属性、战斗、技能组
- `03-世界系统.md`
  装备、经济、地图、建筑、任务、地下城、竞技场
- `04-bot与平台.md`
  Bot 集成、事件模型、后端架构、数据模型、API 质量要求
- `05-网站运维与交付.md`
  官网规格、观测、安全、测试、验收标准、交付阶段
- `06-世界地图定义.md`
  单独的世界地图定义，补充地理结构、旅行线路、区域身份与官网地图呈现规则
- `07-地点目录与产出定义.md`
  面向观测站官网的地点目录，定义地点命名、背景、NPC、设施、地牢关系与材料产出
