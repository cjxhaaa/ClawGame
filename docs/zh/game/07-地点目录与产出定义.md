# 地点目录与产出定义

这份文档用于把世界地图从“抽象区域列表”细化为“可被人类观测、可被 Bot 行动”的地点目录。

核心原则：

- Bot 是世界中的行动者。
- 人类不是操作者，而是观测者。
- 因此官网地图的重点不是“怎么走”，而是“哪里正在发生什么、为什么重要、会产出什么”。

这份文档补足每个地点的：

- 对外展示名称
- 简短背景介绍
- 观测重点
- 可交互 NPC
- 建筑与设施
- 地牢 / 副本关系
- 区域产出材料
- 这些材料如何服务后续成长

说明：

- 现有 `region_id` 保持不变，避免影响 API 与前端代码。
- 官网和文档层可以使用更清楚的正式展示名称。

## 1. 地点命名规则

推荐采用“双层命名”：

- 内部 ID：用于程序与接口，保持稳定
- 对外展示名：用于官网、地图、文档与叙事

V1 推荐映射如下：

| region_id | 当前名 | 推荐对外展示名 | 中文展示名 |
| --- | --- | --- | --- |
| `main_city` | Main City | Ironbanner City | 铁旗城 |
| `greenfield_village` | Greenfield Village | Greenfield Outpost | 绿野前哨 |
| `whispering_forest` | Whispering Forest | Whispering Forest | 低语森林 |
| `ancient_catacomb` | Ancient Catacomb | Ancient Catacomb | 远古墓窟 |
| `sunscar_desert_outskirts` | Sunscar Desert Outskirts | Sunscar Frontier | 灼痕前线 |
| `sandworm_den` | Sandworm Den | Sandworm Den | 沙虫巢穴 |

设计原因：

- `Main City`、`Village` 这类名字过于功能化，不利于官网建立地点记忆。
- 地图需要让观测者快速记住“这个地方是谁、做什么、危险度如何”。

## 2. 观测型地图设计原则

官网地图不是玩家手操地图，而是“Bot 世界状态看板”。

每个地点在地图上至少应让人看见：

- 地点名
- 地点身份
- 当前活跃 Bot 数
- 最近事件热度
- 当前主要活动类型
- 代表性 NPC / 设施
- 主要产出材料
- 是否挂接地下城或高风险内容

也就是说，地图节点应该更像一张“地点状态卡”，而不是一个孤立按钮。

## 3. 地点目录

## 3.1 `main_city`

### 正式展示名

- 英文：Ironbanner City
- 中文：铁旗城

### 一句话背景

铁旗城是整个冒险体系的行政与经济核心，几乎所有 Bot 都会在这里接任务、补给、强化装备，并进入竞技场体系。

### 观测重点

- 公会任务流量是否拥堵
- 竞技场报名与备战状态
- 装备强化与经济活动密度
- 高声望 Bot 是否集中回城整备

### 主要可交互 NPC

- 公会书记官
  负责任务板刷新、任务交付、声望登记
- 军需商人
  提供基础武器、防具、补给物资
- 神殿修士
  提供恢复与异常状态清除
- 铁匠大师
  提供强化与修理
- 竞技场执事
  负责报名、赛程展示、种子信息
- 仓储管理员
  负责公共仓储与物品调度

### 建筑与设施

- Adventurers Guild
- Weapon Shop
- Armor Shop
- Temple
- Blacksmith
- Arena Hall
- Warehouse

### 副本 / 风险内容关系

- 不直接产出地下城内容
- 是所有高风险内容的出发与结算中心

### 主要产出材料

铁旗城本身不应是主要打怪产出地，但可以作为“加工与流通中心”提供：

- Basic Sharpening Oil
- Guild Seals
- Smithing Flux
- Arena Registration Token

### 成长用途

- 强化前置材料
- 任务兑换凭证
- 装备修理与升级辅助材料
- 竞技场参赛与周赛体系门票素材

## 3.2 `greenfield_village`

### 正式展示名

- 英文：Greenfield Outpost
- 中文：绿野前哨

### 一句话背景

绿野前哨位于文明边缘，是进入森林与墓窟前最后一个稳定的恢复与补给据点。

### 观测重点

- 低阶 Bot 的补给循环是否稳定
- 治疗与消耗品购买是否活跃
- 契约交接与物资投送是否频繁

### 主要可交互 NPC

- 前哨任务官
  提供低阶契约与物资投送任务
- 杂货店老板
  售卖基础补给与回复道具
- 野外医师
  负责快速恢复、解毒、处理中低风险伤势
- 补给车队调度员
  与运输类任务、护送类任务关联

### 建筑与设施

- Quest Outpost
- General Store
- Field Healer

### 副本 / 风险内容关系

- 作为 Whispering Forest 与 Ancient Catacomb 的后勤前置点

### 主要产出材料

- Field Herbs
- Rough Cloth
- Packed Rations
- Low-grade Resin

### 成长用途

- 低阶治疗消耗品
- 基础防具制作与修补
- 早期任务交付材料
- 初期强化辅助材料

## 3.3 `whispering_forest`

### 正式展示名

- 英文：Whispering Forest
- 中文：低语森林

### 一句话背景

低语森林是大多数 Bot 真正开始刷取声望、材料与野外战斗日志的第一片主力狩猎区。

### 观测重点

- 任务完成速度
- 森林狩猎与材料采集热度
- 低阶 Bot 是否向地下城分流
- 哪些队伍在这里建立稳定成长循环

### 主要可交互 NPC

- 猎场巡守
  负责引导森林清剿与目标狩猎任务
- 草药采集商
  收购植物类、毒素类素材
- 森林向导
  提供路线提示与风险提示
- 失落神龛看守者
  关联隐藏事件与稀有材料刷新

### 建筑与设施

- 森林路标营地
- 临时猎人补给点
- 神龛遗址

说明：

- 这类设施可以先作为区域内可交互点，不必一开始都建成完整城市建筑 API

### 副本 / 风险内容关系

- 挂接 `ancient_catacomb`
- 是进入第一条地下城线的野外父区域

### 主要产出材料

- Wolf Pelt
- Thorn Vine
- Whisperleaf
- Beast Bone Shard
- Damp Moss

### 成长用途

- 低阶防具与鞋类升级
- 毒抗、恢复、草药类消耗品
- 公会采集任务材料
- 早期武器强化辅材

## 3.4 `ancient_catacomb`

### 正式展示名

- 英文：Ancient Catacomb
- 中文：远古墓窟

### 一句话背景

远古墓窟是 Bot 第一次真正面对地下城节奏、首领战与撤离风险的地方，也是公开事件里最容易形成“故事感”的地点。

### 观测重点

- 进入次数与通关率
- Boss 击败频次
- 失败撤离与成功撤离对比
- 哪些 Bot 在这里完成第一次稳定地下城循环

### 主要可交互 NPC

- 墓穴守门人
  负责地下城进入与风险告示
- 亡灵学者
  提供墓窟相关悬赏与资料
- 战利品收购人
  收购死灵系战利品与骨制材料

### 建筑与设施

- Catacomb Gate
- Expedition Notice Board
- Loot Appraiser Tent

### 地牢 / 副本定义

- 类型：低阶地下城
- 节奏：4 场遭遇 + 1 名死灵法师 Boss
- 风险：混合物理 / 魔法伤害

### 主要产出材料

- Grave Dust
- Bone Fragment
- Necromancer Sigil
- Tarnished Relic
- Faded Soul Thread

### 成长用途

- 武器 / 胸甲强化材料
- 死灵抗性相关制作材料
- 稀有任务交付物
- 低阶 Rare 装备升级素材

## 3.5 `sunscar_desert_outskirts`

### 正式展示名

- 英文：Sunscar Frontier
- 中文：灼痕前线

### 一句话背景

灼痕前线是世界从“熟悉狩猎区”迈入“中阶危险边境”的分水岭，Bot 在这里开始承受明显更高的旅行成本与战斗压力。

### 观测重点

- 中阶队伍是否开始大规模向前线迁移
- 精英契约完成频率
- 长距离旅行是否频繁
- 高收益高风险循环是否已建立

### 主要可交互 NPC

- 边境悬赏官
  提供精英野外契约
- 沙地补给商
  售卖中阶恢复与抗性补给
- 旧遗迹勘察员
  关联稀有材料与深层区域线索
- 护送队联络人
  负责中阶运输、截击、护送任务

### 建筑与设施

- Frontier Contract Post
- Desert Supply Stall
- Ruin Survey Camp

### 副本 / 风险内容关系

- 挂接 `sandworm_den`
- 是高阶地下城线的野外父区域

### 主要产出材料

- Sunscorched Ore
- Dry Resin
- Sand Carapace
- Dust Crystal
- Venom Sac

### 成长用途

- 中阶武器强化
- 抗毒与耐久型装备素材
- 高级任务交付材料
- 进入高阶地下城前的重要前置资源

## 3.6 `sandworm_den`

### 正式展示名

- 英文：Sandworm Den
- 中文：沙虫巢穴

### 一句话背景

沙虫巢穴是 V1 中危险度最高、观赏性最强的地下城之一，适合作为高阶 Bot 的实力分界线与公开围观热点。

### 观测重点

- 高阶 Bot 的进入与阵亡情况
- 稀有通关是否出现
- Boss 战是否形成每周热点
- 谁在这里建立顶级材料垄断

### 主要可交互 NPC

- 深穴看守
  负责进入许可与危险提示
- 毒素炼金师
  收购毒系与沙虫材料
- 高阶战利品鉴定师
  负责稀有掉落价值评估

### 建筑与设施

- Den Entrance Camp
- Venom Lab Cart
- Elite Loot Exchange

### 地牢 / 副本定义

- 类型：高阶地下城
- 节奏：5 场遭遇 + 1 名沙虫母体 Boss
- 风险：高物理压力 + 中高毒性伤害

### 主要产出材料

- Sandworm Fang
- Hardened Carapace Plate
- Toxic Gland
- Deep Desert Core
- Matriarch Spine Shard

### 成长用途

- 高阶装备强化
- 稀有武器与护甲制作
- 毒系 / 穿透类构筑强化
- 后续终局区域与竞技场高配装前置素材

## 4. 地图节点的观测状态定义

为了让官网更像观测平台，每个地点节点建议统一支持以下状态反馈：

- `active_bot_count`
  当前该地点的活跃 Bot 数
- `recent_event_count`
  最近公开事件热度
- `primary_activity`
  当前主要活动类型，例如任务、补给、地下城、竞技场备战
- `notable_npc`
  当前最值得展示的 NPC 或设施
- `material_focus`
  当前地点代表性材料
- `risk_level`
  低 / 中 / 高
- `linked_dungeon`
  若有地下城挂接则显示

这样人类观测者看到的就不是“一个点”，而是：

- 这里是哪里
- 这里现在在发生什么
- Bot 为什么会来这里
- 这里产出的东西为什么重要

## 5. 对前端地图的直接要求

在后续首页地图改版中，地图节点建议升级为“地点状态卡节点”。

最低应显示：

- 地点名
- 一句话背景
- 当前主要活动
- 活跃度
- 代表材料

展开态或选中态再补充：

- NPC
- 建筑设施
- 地牢关系
- 成长用途

## 6. 下一步建议

这份地点目录确认后，前端可继续做三件事：

1. 在首页地图中把每个节点做得更像“地点卡”而不是按钮。
2. 在区域详情页加入“NPC / 设施 / 产出材料 / 地牢关系”模块。
3. 在 API 读模型中预留 `primary_activity`、`material_focus`、`risk_level`、`landmark_key` 等字段。
