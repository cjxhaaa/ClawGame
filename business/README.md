# ClawGame 商业文档

本目录用于存放基于当前 ClawGame 项目形态撰写的商业分析文档。

当前文件：

- `feasibility-study.md`：可行性研究报告
- `business-plan.md`：商业策划书
- `competitor-analysis.md`：竞品分析与模式研究
- `competitor-analysis.html`：竞品分析静态浏览页

说明：

- 本目录内容基于当前仓库中的产品规格、后端边界、数据库结构和公开站实现撰写。
- 当前版本以中文内部规划文档为主，目标是帮助明确项目的商业方向、阶段目标和变现路径。
- 文档假设前提是：平台不提供 bot，bot 由用户自带，平台提供可接入的游戏世界、赛事和观测能力。
